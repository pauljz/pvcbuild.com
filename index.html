Get started with Chocolatey.

cinst pvc

Install some plugins.

pvc install Pvc.Less

Then create a pvcfile.csx

pvc.Task("default", () => {
    pvc.Source("*.less", "*.sass")
       .PipeIf("less$", new PvcLess())
       .PipeIf("(sass|scss)$", new PvcSass())
       .Save("~/deploy");
});

And let the magic happen:

pvc

Check out the documentation to learn more about how to build your pvcfile.

Plugins

Plugins provide the functionality that makes PVC nice to use. Like we saw in the Getting Started section, you can install plugins from the command line using pvc install Pvc.PluginName

Here's a selection of PVC plugins
Pvc.MSBuild Build your project with PVC.
Pvc.NuGet Build and push packages to NuGet and SymbolSource.
Pvc.Less Compiles LESS to CSS and creates source maps.
Pvc.Sass Compiles SASS to CSS and creates source maps.
Pvc.Ajaxmin Minifies JavaScript and generates sourcemaps.
Pvc.HtmlCompressor Minifies HTML.
Pvc.S3 Upload files to Amazon S3.
Pvc.AzureBlob Upload files to Microsoft Azure Blob Storage.
Pvc.Browserify Runs Browserify (yep, from Node!) on your JS.
